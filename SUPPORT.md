# Support

`elementor2gads` is community-supported open-source software maintained by
[Elasyn](https://elasyn.com.au).

Before requesting help:

1. Read the [README](README.md) and relevant pages in `docs/`.
2. Confirm the problem occurs with the latest release.
3. Reproduce it with a small, entirely synthetic CSV.
4. Search existing issues before opening a new one.

Use the bug report template for reproducible defects and the feature request template for proposed
improvements. Public support is provided on a best-effort basis without a response-time guarantee.

Never attach or paste real form submissions, Customer Match exports, email addresses, phone
numbers, names, IP addresses, credentials, or confidential business information. Sanitising a
filename is not enough; replace every record with synthetic data.

Report suspected vulnerabilities privately according to [SECURITY.md](SECURITY.md). Commercial or
professional-service enquiries can be directed to Elasyn through
[elasyn.com.au](https://elasyn.com.au).
